# Contributing

hoardodile is maintained by a single owner at a personal pace. **Issues are welcome; pull requests are not accepted.**

## Reporting a bug

Open an issue and include:

- Environment: OS, Node.js version, and the hoardodile version/commit
- Steps to reproduce
- What you expected vs. what actually happened
- Relevant logs (redact anything private first)

## Requesting a feature

Open an issue describing the use case and the problem it solves. Concrete scenarios are far more useful than abstract ideas.

## Security issues

Please do **not** file a public issue for security vulnerabilities. Report them privately through [GitHub Private Vulnerability Reporting](https://github.com/wooloo26/hoardodile/security/advisories/new) — see [SECURITY.md](.github/SECURITY.md).
